# Shader Image Transition

A smooth image transition effect built with Three.js and WebGL shaders.

## Description

This project demonstrates a custom shader-based image transition effect using Three.js. It features a gallery of images with smooth transitions triggered by clicking on thumbnails. The shader creates a wavy, pixelated transition effect with customizable parameters.

## Features

- Custom WebGL shader for smooth image transitions
- Thumbnail-based navigation
- Responsive design
- Uses external image assets from CDN

## Technologies Used

- Three.js
- WebGL Shaders (GLSL)
- JavaScript
- HTML5
- CSS3

## How to Use

1. Clone the repository
2. Open `index.html` in a web browser
3. Click on thumbnails to transition between images
